parse_files()¶

Takes the list of data_files stored in the controller and determines:

  • self.experiment_date from the first ccnc file record

  • ccnc_data from the csv file(s)

  • smps_data from the txt file

  • Sets the counts_to_conc_conv value by finding the CPC Sample Flow(lpm) value

from the SMPS file and performs the following conversion to convert the liter per minute measure into a second per CC value.

(1 minute / “CPC Sample Flow” L) * (60 seconds / 1 minute) * (1 L / 1000 cc)

create_scans()¶

Creates the scans objects and updates via the following scan methods:

  • set_start_time From the SMPS file

  • set_end_time Start time + Duration

  • set_up_time From the SMPS file

  • set_down_time From the SMPS file (retrace time)

  • set_duration Scan up time + Scan down time

  • set_counts_2_conc As calculated earlier

  • set_cpc_sample_flow from the SMPS file

get_normalized_concentration()¶

Updates the scans objects via the following scan methods:

  • add_to_diameter_midpoints The diameter midpoints

  • add_to_raw_normalized_concs # RESEARCH In English?

Normalized flow_rate is also called dN/dLogDp, which is the notation used in the graph. For further information see tsi documentation.
