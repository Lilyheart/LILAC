scan_index_changed(new_scan_index)¶

Callback function set when a new scan is selected on the Scan Information docker widget. It:

  • Sets the current scan index in the controller (set_scan_index)

  • Switches to the new scan via the controller (switch_to_scan)

Parameters

new_scan_index (int) – The new index to set in the the controller and view.

shift_factor_changed(new_shift_index)¶

Callback function set when the shift factor is changed on the Scan Information docker widget. It:

  • Sets the shift factor on the scan (set_shift_factor)

  • Reprocesses the data (generate_processed_data)

  • Reloads the current scan into the display (switch_to_scan)

Parameters

new_shift_index (int) – The new shift value to set in the controller
